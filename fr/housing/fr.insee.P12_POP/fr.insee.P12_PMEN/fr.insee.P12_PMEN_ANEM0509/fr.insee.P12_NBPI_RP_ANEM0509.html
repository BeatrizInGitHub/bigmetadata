Pièces Rés princ Mén. emménagés entre 5-9 ans¶

Nombre de pièces des résidences principales dans lesquelles le ménage a emménagé entre 5 et 9 ans

Per Pièces rés princ
Per sq km
Raw count

Update a field of a table with an estimate of “Pièces Rés princ Mén. emménagés entre 5-9 ans” per Pièces rés princ for a polygon using a combination of weighting by area and Pièces rés princ.

WITH data AS (
  SELECT cartodb_id AS id, the_geom FROM <table_to_update>
),
meta AS (
  SELECT OBS_GetMeta(ST_SetSRID(ST_Extent(the_geom), 4326),
  '[{"numer_id": "fr.insee.P12_NBPI_RP_ANEM0509",
    "denom_id": "fr.insee.P12_NBPI_RP",
    "normalization": "denominated"}]'
  , 1, 1) meta
FROM data)
UPDATE <table_to_update> SET <field_to_update> = (data->0->>'value')::Numeric
FROM OBS_GetData(
      (SELECT ARRAY_AGG((the_geom, id)::geomval) FROM data),
      (SELECT meta FROM meta)) obs_data
WHERE <table_to_update>.cartodb_id = obs_data.id

Obtain an estimate of “Pièces Rés princ Mén. emménagés entre 5-9 ans” per Pièces rés princ for a polygon using a combination of weighting by area and Pièces rés princ.

WITH data AS (
  SELECT 1 AS id,
  ST_Buffer(ST_SetSRID(ST_Point(2.361, 48.86), 4326), 0.01) the_geom
),
meta AS (
  SELECT OBS_GetMeta(ST_SetSRID(ST_Extent(the_geom), 4326),
  '[{"numer_id": "fr.insee.P12_NBPI_RP_ANEM0509",
     "denom_id": "fr.insee.P12_NBPI_RP",
     "normalization": "denominated"}]'
  , 1, 1) meta
FROM data)
SELECT id, (data->0->>'value')::Numeric AS p12_nbpi_rp_anem0509_rate_2012
FROM OBS_GetData(
       (SELECT ARRAY_AGG((the_geom, id)::geomval) FROM data),
       (SELECT meta FROM meta))
